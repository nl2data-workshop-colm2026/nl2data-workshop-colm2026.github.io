# NL2Data Workshop (CoLM 2026)

Simple static website for the NL2Data workshop.

## Files
- `index.html`: page structure and workshop content
- `styles.css`: visual style and responsive layout

## Local preview
Run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Publish with GitHub Pages
1. Push this repo to GitHub.
2. In repo settings, enable **Pages**.
3. Set source to the `main` branch and `/ (root)`.
